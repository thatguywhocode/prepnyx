# 🚀 Prepnyx

AI-powered study platform that helps students plan, track, and optimize their preparation using personalized recommendations.

## 🏗️ Tech Stack
- React.js
- Node.js (Fastify)
- MongoDB
- FastAPI (AI)

## 📦 Features
- Authentication & User Management
- Study Planner
- Progress Tracking & Analytics
- Recommendation Engine
- Notes & Revision
- Smart Notifications
- GenAI Assistant
